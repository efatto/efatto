Outgoing Email Extension

Configure userwise outgoing email server


By default in Odoo, when email goes out from system and if no outgoing mail server specified, it selects the outgoing server which is having low priority. From any user if you try to send email, system will select only one lowest prioritised outgoing server. After installation of this module, you can configure user wise outgoing server. So it will be possible to send user wise outgoing emails from Odoo. With this module, you can use multiple outgoing servers depending on the user that is sending an email.

Setup & Configure

To be able to assign user to outgoing server, you need to select an option "Send userwise outgoing emails ?" under menu Settings >> Cofiguration >> General Settings.

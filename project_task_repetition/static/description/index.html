Project task repetition

Create 'recurrent' tasks

Tired of keep having to duplicate/edit tasks whenever you want the same thing to be done over and over?

This module will create independent tasks that are copies of an original one but with different deadlines.

You can freely change the new tasks.

Easy to use

The way to set a task repetition is pretty much the same as when you create a recurrent event in the calendar.

You get to choose several conditions that will cause the tasks to be created with different deadlines.

Examples:

  • set a daily task for the next 100 days
  • set 3 repetitions, each with a 2 month interval, on tuesdays

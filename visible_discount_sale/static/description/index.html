Visible Discount in Sale

Enable discount visibility separately on Sale Order from pricelist

Visible Discount on Sales Order

By installing this module, you will find an option under Settings >> Configuration >> Sales menu.

Visible Discount in Pricelist

Once you tick the option "Visible Discount on Sales Order" under Settings >> Configuration >> Sales menu, a field named as "Visible Discount" will be visible in any Pricelist form. That field will be visible only if your pricelist type will be kind of "Sale Pricelist".

Separated Base price & Discount in Sale Order

In the sale order, if pricelist has "Visible Discount" ticked then Base price and discount will be shown as separated value inside order line.This separation will be applicable on the final price calculated after apply through pricelist. However if you want to increase/decrease discount in order line then you can set afterwards in order line.

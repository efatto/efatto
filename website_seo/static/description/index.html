Default Parameters

Required Parameter

  • Attribute Separator.
  • Value Separator.
  • Meta Title Size.
  • Meta Desc. Size.
  • Meta Keyword Size.
  • Meta Keywords.

SEO For Product

This module provide the essentials support in managing the SEO for Products

First of all you will have to generate the SEO Meta -Info for you Product for this just go through Sale-->Website SEO

  • Select Product in the SEO For List.
  • Select product attribute for managing the meta Title,Description, Keywords .
  • Assign/Change the Multi Attribute Separator.
  • Assign/Change the Multi Value Separator.
  • Click on Generate Now.

Fields For Product SEO

  • Product Name.
  • Category Name.
  • Description.
  • Product Description .
  • EAN13 Bar-code .
  • Company Name.

After the Clicking the generate button the you will see the message about the Updation, message also include number of effected product with this SEO generation/Updation.

Fields For Category SEO

  • Parent Category.
